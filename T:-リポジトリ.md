# README
リポジトリタイプによってやることが異なるので、大まかにタイプを分けてそれぞれの項目にある程度最適化を意識している。

## テンプレート
```
## リポジトリ名

## できること
- PR

## できないこと
[専用リポジトリ](https://github.com/shimajima-eiji/__Github-Operation/issues)で一括管理しているため、以下機能をリポジトリに持たない。
- Issue
- Wiki
- Project
- Actions

## テンプレートバージョン
v2022.01.06
```
